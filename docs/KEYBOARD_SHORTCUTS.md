# Keyboard shortcuts

MedQuiz is designed for mouse and keyboard study sessions. Shortcuts are disabled while modifier keys (Ctrl, Alt, Meta) are held so they do not conflict with system shortcuts.

## Quiz sessions

| Key | Action |
|-----|--------|
| `1` – `5` | Select answer option A – E |
| `Enter` | Submit answer, advance to next question, or view results |
| `←` | Previous question |
| `→` | Next question (when allowed by answer timing) |
| `Esc` | Exit quiz — opens a confirmation dialog |

**Tip:** Click anywhere on the quiz page if shortcuts stop responding; the page must have focus (`tabindex` on the quiz container).

## Learn sessions

| Key | Action |
|-----|--------|
| `1` – `5` | Select option (Multiple Choice and Analogy Completion only) |
| `Enter` | Check answer, or advance to the next item |
| `Esc` | Exit learn session — opens a confirmation dialog |

Text-based modalities (short answer, create analogy, matching, arrows) use `Enter` to check and advance once inputs are complete.

## Confirmation dialogs

Used when leaving a session, deleting questions, or deleting source files.

| Key | Action |
|-----|--------|
| `Esc` | Cancel |
| `Enter` | Confirm |

## First-run onboarding

| Key | Action |
|-----|--------|
| `Enter` | Next step |
| `Esc` | Back one step, or dismiss on the welcome step |

## Navigation

Sidebar navigation (Home, Materials, Question Bank, Settings) is mouse-driven. The sidebar is locked during active quiz or Learn sessions.

## In-app reference

Open **Settings** and scroll to **Keyboard shortcuts** for a quick table inside the app.
